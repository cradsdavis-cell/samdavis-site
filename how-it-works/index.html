How it works

What working with me actually looks like.

No deck, no theatre. Here's the shape of a Coaching Block from the first email to graduation.

The 4-session arc

Most clients come in via a Coaching Block — four 60-min 1:1 sessions over four weeks.

Session 1 — install + foundation

Stack working on your machine (Claude Code + VS Code + Obsidian). I sit beside you while we wire it up. You leave with a working EA skeleton.

Session 2 — first build

You pick one workflow that's eating your week. We build the skill that handles it. You operate it live; we refine together.

Session 3 — second build + integration

Second workflow. Plus integrations to the tools you already use (Gmail, Calendar, Slack, Notion, whatever).

Session 4 — graduation

We review what's running, what you'd build next, and what to do without me. Most clients continue with a $600/mo retainer; some don't need to.

Onboarding (before Session 1)

You get a personalised onboarding pack 3–4 days before Session 1. It includes:

  • Install doc — what to download + set up before we talk.
  • Your context worksheet — what you do, what's painful, what good would look like.
  • Pre-read on how an AI EA fits a business like yours.

Your first 2 weeks

Front-loaded. Most of the install + first build happens in week 1. Week 2 is consolidation + the second build. Weeks 3 and 4 are integration + graduation. After that, optional Continuation Retainer for ongoing support.
